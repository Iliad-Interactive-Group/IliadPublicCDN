# Voice Samples

Pre-generated voice samples for GrowthOS Studio voice preview.

## Files

30 MP3 files, one per Gemini TTS voice:
- 15 male voices
- 15 female voices

Each sample is ~8-10 seconds of generic broadcast copy.

## Generation

Generated using `tools/generate-voice-samples/` in HomerDev repo.

## Usage

These files are served via CDN at:
`https://cdn.iliadinteractive.com/audio/voice-samples/{voicename}.mp3`

Example: `https://cdn.iliadinteractive.com/audio/voice-samples/puck.mp3`
